1. Introduction to HTML

HTML is used to structure content on the web. It uses tags to tell the browser how to display content.

<!DOCTYPE html>
<html>
<head>
    <title>My First Page</title>
</head>
<body>
    <h1>Hello World</h1>
</body>
</html>
