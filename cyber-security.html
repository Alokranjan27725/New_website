Cyber Security Basics

Introduction

Cyber security ka matlab hai systems, networks aur programs ko digital attacks se bachana. Ye attacks aksar sensitive data churaane, systems ko nuksaan pahunchane, ya unauthorized access ke liye kiye jaate hain.

Key Concepts

  • Confidentiality — Data sirf authorised logon ke liye accessible ho.
  • Integrity — Data accurate aur unchanged rahe jab tak authorised na ho.
  • Availability — Systems aur services jab chahiye tab available rahen.

Common Threats

  • Phishing — Fake emails/websites se login info churaana.
  • Malware — Viruses, trojans, ransomware jo system ko damage karte hain.
  • Man-in-the-Middle (MitM) — Communication ke beech me data intercept karna.
  • Brute-force attacks — Passwords ko guess karne ki koshish.

Practical Best Practices

  1. Strong & unique passwords use karo — password manager ka istemal karo.
  2. Two-Factor Authentication (2FA) enable karo jahan available ho.
  3. Software aur OS ko regularly update karo (patches important hain).
  4. Unknown links aur attachments pe click mat karo — phishing se savdhaan raho.
  5. Public Wi-Fi par sensitive kaam na karo; VPN ka use karo agar zaroori ho.
  6. Regular backups rakho — ransomware se bachne ka best defence.

Quick Tools & Resources
