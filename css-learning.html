CSS Syntax

CSS syntax consists of a selector and a declaration block. The selector points to the HTML element you want to style.

selector {
    property: value;
}

Example:

p {
    color: red;
    font-size: 16px;
}

CSS Selectors

CSS selectors are used to target HTML elements for styling. Common types include:

  • Element Selector - Selects elements by their tag name (e.g., p, h1)
  • ID Selector - Selects an element by its ID using # (e.g., #myId)
  • Class Selector - Selects elements by their class using . (e.g., .myClass)
  • Universal Selector - Selects all elements using *
  • Group Selector - Selects multiple elements by separating selectors with a comma

Example:

/* Element selector */
p {
    color: blue;
}

/* ID selector */
#title {
    font-size: 24px;
}

/* Class selector */
.intro {
    background-color: yellow;
}

CSS Padding and Margin

Padding is space inside the element's border, margin is space outside it.

.box1 {
    padding: 10px;        /* All sides */
    margin: 15px;         /* All sides */
}
.box2 {
    padding: 10px 20px;   /* Top-Bottom | Left-Right */
    margin: 5px 15px;     
}
.box3 {
    padding: 5px 10px 15px 20px;  /* Top | Right | Bottom | Left */
    margin: 0 auto;               /* Center horizontally */
}

}

    padding: 20px;

CSS Transitions

Smoothly change property values.

.transition-box {
    width: 100px;
    height: 100px;
    background: orange;
    transition: background 0.5s, transform 0.5s;
}
.transition-box:hover {
    background: red;
    transform: scale(1.2);
}

CSS Animations

Create moving effects using keyframes.

@keyframes move {
    0%   { transform: translateX(0); }
    50%  { transform: translateX(100px); }
    100% { transform: translateX(0); }
}
.animate-box {
    width: 100px;
    height: 100px;
    background: purple;
    animation: move 2s infinite;
}

CSS Variables

Re-use values using custom properties.

:root {
    --main-color: #3498db;
    --padding-size: 20px;
}
.box {
    background: var(--main-color);
    padding: var(--padding-size);
}

CSS Pseudo-classes

Style elements based on their state.

a:hover {
    color: red;
}
input:focus {
    border: 2px solid blue;
}
li:first-child {
    font-weight: bold;
}

CSS Media Queries

Make designs responsive for different screen sizes.

@media (max-width: 600px) {
    body {
        background-color: lightgray;
    }
    .box {
        width: 100%;
    }
}
